Case Study

QuantPilot

面向量化研究的 AI 工作台，把策略想法、数据分析、可视化、结果追踪和 Agent 协作放进一个可复用的研究环境里。

查看仓库 更多项目
Problem

投研不是一次对话，而是一条可追踪的工作流

量化研究涉及数据源、清洗、特征、策略假设、回测结果和复盘。QuantPilot 的方向是让 Agent 帮忙推进分析，但把中间过程留在结构化工作台里。

My Role

把数据工具和 AI 助手做成产品界面

这个项目强调产品化能力：不是只做脚本，而是为研究流程设计可交互界面、模块边界和结果沉淀方式，让后续迭代有明确入口。

核心能力

  • 把策略想法转成可执行分析任务。
  • 用数据视图承载行情、指标、回测和比较结果。
  • 将 Agent 的推理过程和产出沉淀为工作区资产。
  • 为投研流程保留可复盘的上下文和结果链路。

展示价值

QuantPilot 适合作为“AI + 数据产品”的代表项目：它能展示你如何把 Agent 能力嵌进复杂业务流程，而不是停留在聊天框或一次性脚本里。
